# Chris Duchene — Hospitality & Casino Data Analytics Portfolio

This portfolio demonstrates how I turn messy operational and financial data into executive sready insights.

## Focus Areas
- Revenue integrity (slots, tables, hotel, F&B)
- Promotional effectiveness
- Forecasting & variance analysis
- Labor vs volume optimization
- Player and guest segmentation

## Tool Stack
- PostgreSQL + DBeaver
- Power BI (PL-300 Certified)
- Python (pandas, numpy)
- Excel / Power Query
- GitHub

## Project Framework
Each business problem is solved three ways:
1) SQL — data extraction, QA, validation
2) Power BI — modeling, dashboards, KPIs
3) Python — analysis, forecasting, automation
